Pull Shark
Medium ⏱ 15 mins ✅ Solo
Run these commands in your terminal
# Step 1: Clone any of your repos
git clone https://github.com/sahilansari0002/CODSOFT.git
cd CODSOFT
# Step 2: Create a new branch
git checkout -b badge/pull-shark-1
# Step 3: Make any small change
echo "# Pull Shark Badge" >> BADGE.md
git add BADGE.md
git commit -m "feat: add badge notes"
# Step 4: Push the branch
git push origin badge/pull-shark-1
5
Go to GitHub → you'll see a prompt to "Compare & pull request" → click it
6
Click "Create pull request" → then "Merge pull request"
7
Repeat steps 2–6 one more time with branch name badge/pull-shark-2
8
After 2 merged PRs → 🦈 Pull Shark badge unlocked!

YOLO
Medium ⏱ 20 mins
⚠️ Needs: 2 GitHub accounts (yours + a friend or alt account)
1
Account B (friend/alt): Create a new repo on GitHub, go to Settings → Collaborators → Add people → add sahilansari0002
2
Account A (you): Accept the invite from your email or notifications
# Account A: Clone the repo, make a branch, push
git clone https://github.com/ACCOUNT_B/REPO_NAME.git
cd REPO_NAME
git checkout -b yolo-branch
echo "YOLO" >> yolo.md
git add . && git commit -m "yolo commit"
git push origin yolo-branch
3
Go to GitHub → Open a Pull Request from yolo-branch → main
4
Merge it WITHOUT requesting any review — just click Merge directly
5
🤪 YOLO badge unlocked for the account that merged!
⚠️ Make sure the repo has no branch protection rules that require reviews. Check: Settings → Branches → no rules on main.

Galaxy Brain
Medium ⏱ Days–Weeks ✅ Solo
1
Visit github.com/orgs/community/discussions
2
Filter by "Unanswered" to find open questions
3
Write a clear, detailed, helpful answer — don't post one-liners
4
When the author marks your reply as ✅ Answered → that counts!
5
Get 2 accepted answers → badge unlocked!
💡 Pro tip: Answer questions about topics you know well — Android, Flutter, Firebase, Java. Your expertise = faster accepted answers.
