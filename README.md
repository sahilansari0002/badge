# 🏆 GitHub Badge Hunter — @sahilansari0002

A stylish and interactive **GitHub badge guide** that helps users unlock popular GitHub profile badges with **step-by-step instructions**, **copy-paste commands**, and a **progress tracker**.

---

## ✨ Features

- ⚡ **Quick Wins** section for easy badges
- 🦈 **Pull Shark** badge walkthrough
- 🤪 **YOLO** badge guide
- 🧠 **Galaxy Brain** earning strategy
- 👥 **Pair Extraordinaire** instructions
- ✅ Interactive **Badge Tracker**
- 📋 **Copy command buttons**
- 📊 Progress bar for completed badges
- 🎨 Clean **GitHub-inspired dark UI**

---

## 🌐 Deploy on GitHub Pages

You can host this guide for free using **GitHub Pages**.

### Steps:
1. Upload the project to a GitHub repository
2. Go to **Settings**
3. Click **Pages**
4. Under **Source**, select:
   - Branch: `main`
   - Folder: `/root`
5. Click **Save**

Your site will be live at:

```bash
https://your-username.github.io/repository-name/
```

## 🧩 Badge Guides Included

### 🔫 Quickdraw
Earned by opening and closing an issue quickly.

### 💖 Public Sponsor
Earned by sponsoring a developer publicly.

### 🦈 Pull Shark
Earned by getting pull requests merged.

### 🤪 YOLO
Earned by merging a PR without review.

### 🧠 Galaxy Brain
Earned by providing accepted answers in GitHub Discussions.

### 👥 Pair Extraordinaire
Earned through co-authored commits.

### ⭐ Starstruck
Earned when your repository gets stars.
